# Hairdo Beauty Salon Website

Selamat datang di repository **Hairdo Beauty Salon Website**. Proyek ini adalah website untuk salon kecantikan "Hairdo Beauty" yang menampilkan berbagai layanan kecantikan seperti perawatan rambut, layanan pengantin, make-up, dan portofolio.

## Fitur Utama

- **Halaman Beranda (Home):** Menampilkan informasi singkat mengenai salon.
- **Layanan (Services):** Informasi detail tentang layanan yang ditawarkan.
- **Portofolio (Portfolio):** Galeri layanan dan karya yang pernah dilakukan.
- **Testimonial:** Ulasan pelanggan tentang layanan kami.
- **Kontak (Contact):** Formulir untuk pelanggan yang ingin menghubungi atau membuat janji.


## Teknologi yang Digunakan

- **HTML5**: Untuk struktur dan konten website.
- **CSS3**: Untuk styling dan layout website.
- **Bootstrap**: Untuk desain responsif.
- **JavaScript**: Untuk fitur interaktif.
- **AOS**: Untuk animasi elemen ketika di-scroll.
- **Google Fonts**: Untuk tampilan font modern.

## Cara Menjalankan Proyek

1. **Clone Repository**

2. **Buka File**
- Navigasikan ke folder proyek, lalu buka file `index.html` menggunakan browser favorit Anda.

3. **Atur Assets**
- Pastikan folder `assets` memiliki semua file pendukung (CSS, JS, gambar) di tempat yang tepat.

## Halaman Website

- **Header:** Navigasi untuk berpindah antarbagian seperti Home, Services, Portfolio, Testimonials, dan Contact.
- **Hero Section:** Bagian pembuka dengan informasi utama tentang salon.
- **About Us:** Penjelasan tentang keunggulan layanan salon.
- **Layanan Kami (Services):** Daftar layanan yang ditawarkan dengan deskripsi singkat.
- **Portfolio:** Galeri karya dan layanan.
- **Testimonial:** Ulasan pelanggan yang puas.
- **Contact:** Formulir untuk menghubungi salon.
